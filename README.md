## Getting Started

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.

### Build for Production

```bash
# Build the application
npm run build

# Start production server
npm start
```
